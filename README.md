# Bayna Yadayk — Boek 1 Deel 2 · Examen-Sprint

Gratis interactieve leer-app voor de schrijftoets van **Al-Arabiyyah Bayna Yadayk, Boek 1 Deel 2** (Deel 1 als basis).

👉 **Open de app:** https://ferosh.github.io/bayna-yadayk/

Werkt op telefoon en laptop, ook offline nadat de pagina één keer geladen is.

## Wat zit erin
- 4-uur sprintplan
- Diagnose Deel 1
- Woordkaarten (Deel 2, met voorbeeldzinnen)
- Schrijfmotor (zinspatronen)
- Grammatica: de 9 stencil-onderwerpen + mini-quizzes
- Verleden tijd vervoegen (+ bonus toekomende tijd)
- Spelling (8 valkuilen)
- Quiz (meerkeuze + intypen)
- Schrijftraining met modelverhalen
